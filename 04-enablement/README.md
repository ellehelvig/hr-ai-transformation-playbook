## Delivery guidance

The curriculum is designed for facilitator-led delivery but can be adapted for self-paced use. Module 2's hands-on lab is the highest-value session — don't cut it for time. Module 3 generates the most discussion and sometimes the most pushback; budget flex time.

Recommended delivery cadence: one module per week over 4 weeks, with async practice between sessions.
