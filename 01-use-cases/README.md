# Use case library

Everything in this section is designed to answer one question: **what should we actually build?**

## Contents

| File | Purpose |
|---|---|
| [use-case-library.md](use-case-library.md) | 30+ HR AI use cases organized by function, with AI approach, impact, effort, and risk ratings |
| [prioritization-matrix.md](prioritization-matrix.md) | Scoring framework to rank use cases against your specific context |
| [intake-template.md](intake-template.md) | Submit a new use case for review |

## How to use this section

Start with the [prioritization matrix](prioritization-matrix.md). Score your top 5–8 ideas before committing to any of them. The use case library is a menu, not a prescription.

If you have a use case that isn't in the library, add it via the [intake template](intake-template.md) and open a pull request.
