# Prompt library

Copy-and-adapt prompts for HR workflows. Every prompt is designed as a Copilot tool — AI drafts, a human reviews and owns the output.

## Contents

| File | Prompts included |
|---|---|
| [onboarding.md](onboarding.md) | Personalized onboarding plans, welcome messages, manager checklists, feedback synthesis, buddy matching |
| [performance.md](performance.md) | Review drafts, self-assessments, calibration prep, goal scoring, PIP frameworks |
| [recruiting.md](recruiting.md) | JD review, interview question banks, debrief synthesis, offer letters |

## Prompt quality guidelines

**Before adding a prompt to this library, it should:**
- Have been tested with at least 3 real examples
- Include tuning notes based on what does and doesn't work
- Flag any governance or legal considerations
- Not make autonomous employment decisions
