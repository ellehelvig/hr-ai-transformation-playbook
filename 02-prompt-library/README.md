# Prompt library

Production-ready prompts for HR workflows. Every prompt is structured to be tested, iterated on, and adapted to your organization's voice and policies.

## Contents

| File | Purpose |
|---|---|
| [learning-development.md](learning-development.md) | Prompts for skills gap analysis, learning path generation, manager coaching, and program evaluation |
| [talent-acquisition.md](talent-acquisition.md) | Prompts for inclusive job postings, structured screening, resume evidence extraction, panel debriefs, rejections, and offer equity checks |
| [performance-management.md](performance-management.md) | Prompts for review drafting, calibration prep, PIP documentation, development conversations, and consistency checks |

## How to use these prompts

Each prompt includes the prompt itself, intended use, expected output, and tuning notes from real testing. Treat them as starting points — your organization's policies, tone, and data shape what works.

Before deploying any prompt in a workflow that affects employees, route it through the [risk assessment template](../03-governance/risk-assessment-template.md).
